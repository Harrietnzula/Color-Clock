# Dynamic Color Clock

A React app that displays the current date and time, built with Vite and date-fns.

## How to Run Locally

1. Clone the repository:
   git clone https://github.com/YOUR-USERNAME/color-clock.git

2. Navigate into the project folder:
   cd color-clock

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

5. Open your browser and go to:
   http://localhost:5173

## Built With
- React
- Vite
- date-fns
